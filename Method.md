클래스는 속성을 나타내는 변수와 동작을 나타내는 메소드로 이루어져 있음.

메소드는 클래스에 있는 함수.자바에서 모든 함수는 클래스에 있기 때문에 사실 자바의 모든 함수는 메소드임.

## 기본 문법
메소드의 기본 문법
```java
int sum(int x,int y){
    return x + y;
}
```
sum이라는 메소드를 썼는데 이 함수는 일단 파라미터로 정수형 x와 정수형 y를 받습니다.
그리고 이 메소드는 꼭 정수형을 리턴시켜줘야 함.
그렇지 않으면 오류가 나옴.
return x + y; 는 정수형을 리턴시켜주니까 문제가 없음

만약 리턴값이 없으면 자료형 대신 void라고 써주면 됨. 예를 들어서 예전에 썼던 main 함수도 리턴값이 없기 때문에 void를 사용함.
