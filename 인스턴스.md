클래스가 '설계도'라면 인스턴스는 그 설계도를 따르는 '객체'를 뜻함.
예를 들어 Song 이라는 클래스가 있으면 '걱정말아요 그대','거위의 꿈','다행이다' 등은 Song 클래스를 따르는 인스턴스라고 부를 수도 있고,
편하게는 그냥 Song 인스턴스라고 부를 수 있습니다.

우리의 '은행 계좌 시뮬레이션' 프로그램에서는 Person이라는 클래스가 있는데, '김신의' 라는 이름을 가진 Person 인스턴스, '문종모'라는 이름을 가진 Person 인스턴스,
'성태호' 라는 이름을 가진 Person 인스턴스 등이 있을 수 있겠죠?

실제 인스턴스 생성하는 방법은?

```java
public class BankDriver {
    public static void main(String[] args) {
        // 사람 선언
        Person p1 = new Person();
    }
}
```
이렇게 만들면 p1에는 new 생성자로 새로 생성되었기에 Person에 대한 정보가 들어있다
Person객체에는 저번에 만들었던 변수가 있는데 name, age, cashAmount 변수가 들어있다.
```java
public class BankDriver {
    public static void main(String[] args) {
        // 사람 선언
        Person p1 = new Person();
        p1.name = "김신의";
        p1.age = 28;
        p1.cashAmount = 30000;
    }
}
```
거기에 대한 값을 지정해주면 값까지 들어간 인스턴스까지 생성완료!
