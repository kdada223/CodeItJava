# java-practice

간단히: Java 기초/알고리즘/작은 실습용 코드를 모아두는 연습용 저장소입니다.

사용법
- 각 실습은 src/ 아래에 패키지별로 정리되어 있습니다.
- 예제 실행: javac 와 java 또는 IDE(IntelliJ/VSCode)에서 실행하세요.
  - 컴파일: javac src/.../*.java
  - 실행: java -cp src 패키지.MainClass

폴더 구조(예시)
- src/           — Java 소스 코드
- test/          — 연습용 테스트 코드 (선택)
- notes/         — 학습 메모 및 문제 풀이 정리
- README.md      — 이 파일

커밋 규칙(간단)
- feat: 새로운 문제 풀이 또는 예제 추가
- fix: 버그 수정
- docs: 문서(노트) 업데이트
